Describe the effects of changing the spring constant ks; how does the cloth behave from start to rest with a very low ks? A high ks? The spring constant controls how tight the fabric is. Lower ks values make the fabric much more elastic whereas higher ks values make the fabric more firm.

What about for density? Changing the density is a bit like changing the spring constant. Increases in density thickens the fabric, making it harder to sag whereas lower density allows the fabric to loosely hang. This is due to the increase of weight on the point masses.

What about for damping? Modifying the damping parameter changes the cloth’s ‘speed’, allowing it to move either much faster or slower. Lower damping values display a much more reactive cloth whereas a higher damping value makes it much ‘slower’, much like what we see of Superman’s cape as he descends slowly and menacingly.

Part 5 Deliverables Explain in your own words what is a shader program and how vertex and fragment shaders work together to create lighting and material effects.

A shader program executes on the GPU. It is primarily used to handle rendering effects on a per-pixel basis during real-time applications. There are 2 Parts to a shader, the vertex shader, and the fragment shader. The vertex shader processes each verte of a 3D object then transforms these vertex coordinates from object space into screen space. It outputs final positon of each vertex in screen, while also passing on data such as color, and texture coordiantes. The seocnd part, fragment Shaders deal with processes that actually determines the color and other attributes of each pixel. Fragements are gerneated and outputs the color and depth of each pixel.
